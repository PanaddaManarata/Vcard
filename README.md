# Vcard 
Assignment for Becode.org: 

Create your own virtual vcard using SASS 

## Installation

via sass compiler in visual studio code 


## Usage

```sass
start compiler
make style.scss file

link normal css file in HTML 
   <link rel="stylesheet" href="style.css">
```
